# Triangle on WebGL
## Required packages
1. browserify
2. babel-preset-es2015
3. gl-matrix
4. watchify
## How to build
1. `npm run compile` - compile project
2. `npm run debug` - compile project with debug source to code on ES6
## Control
1. `Arrow keys` - move triangle
2. `]` - increase triangle
3. `[` - decrease triangle
